# **App Name**: EthioConnect

## Core Features:

- User & Business Authentication: Secure email/password and optional phone login, supporting role-based access for Users, Business Owners, and Admin roles.
- User Profile Management: Users can create and edit their profiles including name, bio, and profile picture.
- Post Creation & Chronological Feed: Users and businesses can create text and image posts. All users view a chronological feed mixing user content and business promotions.
- Engagement Features: Users can like and comment on posts, and follow/unfollow other users and businesses.
- Business Profiles & Promotion Tools: Business Owners can create detailed profiles (name, category, description, location via Google Maps, contact info) and post promotional content.
- Ethiopian Payment Integration: Implement payment processing using Telebirr, CBE Birr, and Chapa API to manage business subscription plans (Free, Basic, Premium) and paid features like verified badges or promoted posts.
- AI Promotional Text Generator: A generative AI tool for Business Owners to assist in crafting compelling promotional text for their posts and listings, maximizing engagement.

## Style Guidelines:

- Primary interactive color: An inviting golden-orange (#CD7B15) for vibrancy and engagement, inspired by Ethiopian warmth and community.
- Background color: A subtle, light golden-beige (#FDF5EA) to maintain a clean aesthetic while complementing the primary color.
- Accent color: A rich, warm reddish-orange (#B3301F) for high-contrast elements, calls to action, and highlights, drawing from traditional artisan crafts.
- Headlines and prominent text: 'Poppins', a geometric sans-serif for a modern, precise, and contemporary feel.
- Body text and general UI: 'Inter', a grotesque-style sans-serif for optimal readability and a clean, objective appearance.
- Minimalist line-art icons that are universally understandable and maintain an uncluttered 'Instagram-like' aesthetic, focusing on clarity for mobile-first use.
- A responsive, mobile-first grid-based layout ensuring clean content presentation and intuitive navigation across all screen sizes, reminiscent of modern social feeds.
- Subtle and swift animations for transitions and interactive elements, prioritizing speed and user experience without distracting from the content, aligning with a fast and modern feel.
